Client side/ Server side

voltar

Em HTML e no desenvolvimento web, os termos client-side e server-side referem-se ao local onde o processamento ocorre e como as interações entre o navegador e o servidor são gerenciadas.

O client-side envolve tudo o que é processado diretamente no navegador do usuário. O código que roda no client-side é geralmente escrito em, HTML, CSS e JavaScript

Server-side: O código é executado no servidor, com respostas sendo enviadas de volta ao navegador.

Navegadores

voltar

Em programação, um navegador (ou browser, em inglês) é um software utilizado para acessar e visualizar páginas e aplicações da web. Ele interpreta e renderiza o código escrito em linguagens como HTML, CSS e JavaScript, exibindo as páginas web de maneira visual e interativa para os usuários.
